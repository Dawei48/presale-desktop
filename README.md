# 放心预 - 预售管理系统 v1.0.0

> 桌面版预售管理工具 · 本地数据 · 团队协作 · Word 导出

## 功能

- 🏠 **仪表盘** — 产品/订单统计概览，最近订单
- 📦 **产品管理** — 增删改查，品牌/分类/价格/库存/成本价
- 📋 **订单管理** — 增删改查，状态流转（待确认→已确认→已发货→已完成）
- 🔍 **全局搜索** — 跨产品、订单即时搜索
- 👥 **团队管理** — 管理员/成员角色，密码保护
- 📄 **Word 导出** — 一键生成订单汇总报告 (.docx)

## 快速开始

### 方式一：直接运行 Python

```bash
# 安装依赖
pip install -r requirements.txt

# 启动
python main.py
```

### 方式二：打包成 .exe

```bash
# Windows 双击运行
build.bat
```

打包完成后在 `dist/` 目录找到 `放心预.exe`，可直接分发给其他电脑使用。

## 默认账号

| 用户名 | 密码 | 角色 |
|--------|------|------|

## 数据存储

所有数据存储在程序目录下的 `data/presale.db` (SQLite)，备份只需复制该文件。

## 技术栈

- Python 3.10+
- customtkinter (现代 UI)
- SQLite (本地数据库)
- python-docx (Word 导出)
- PyInstaller (打包 .exe)
