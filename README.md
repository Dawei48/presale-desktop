# 放心预 - 预售管理系统

桌面端预售管理工具，支持多机协作，数据云端同步。

## 功能

- **三级管理**：品牌 → 产品 → 订单
- **云端同步**：基于 Supabase，多台电脑数据实时同步
- **团队协作**：管理员创建账号，成员登录即可查看和操作
- **产品图片**：上传产品图，自动压缩，云端存储
- **订单管理**：新增/编辑/删除订单，快捷切换状态（未发货/已发货/已完成/已取消）
- **日期选择**：日历弹窗选择预售日期
- **全局搜索**：按产品名、客户名、订单号搜索
- **Word 导出**：按品牌导出未发货订单，自动合并同一产品，压缩图片
- **操作日志**：记录所有关键操作

## 下载

前往 [Releases](https://github.com/Dawei48/presale-desktop/releases) 下载最新版 `presale-desktop.exe`。

双击运行，无需安装 Python 环境。

## 使用流程

1. 首次打开 → 创建管理员账号
2. 登录 → 创建品牌 → 添加产品 → 新增订单
3. 其他电脑下载同一个 exe → 注册成员账号 → 数据自动同步
4. 导出 Word 文档用于跟供应商/工厂对接

## 技术栈

- Python 3.11+
- Customtkinter（UI 框架）
- Supabase（云端数据库 + 文件存储）
- python-docx（Word 导出）
- PyInstaller（打包 .exe）
