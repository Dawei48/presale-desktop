# 放心预 - 预售管理系统

> 桌面版预售管理工具 · 本地数据 · 团队协作 · Word 导出

## ✨ 功能

### 🔐 账号管理
- 首次使用创建管理员账号
- 管理员邀请团队成员
- 成员/管理员角色区分

### 📦 三级管理：品牌 → 产品 → 订单
- **品牌管理**：创建、编辑、删除品牌
- **产品管理**：每个品牌下管理产品（名称、价格、产品图片）
- **订单管理**：每个产品下管理订单（订单号、客户、数量、备注、日期）

### 🖼️ 产品图片
- 添加/编辑产品时上传本地图片
- 产品列表缩略图预览
- 图片存储在本地 `data/images/` 目录

### 📋 订单功能
- 📅 **日期选择器**：日历弹窗选择下单日期
- 🔄 **状态快捷切换**：一键切换未发货/已发货/已完成/已取消
- 🔍 **搜索 + 筛选**：按关键词搜索、按状态筛选

### 📄 Word 导出
- 按品牌导出 `.docx` 文件
- 导出内容包含：**产品图片 + 产品名 + 单价 + 预售总数量 + 订单明细表**
- 统计摘要（订单总数、总数量、各状态数量）

## 🚀 下载使用

### 方式一：下载 .exe（推荐）
1. 前往 [Releases](https://github.com/Dawei48/presale-desktop/releases) 下载最新 `presale-desktop.exe`
2. 双击运行，无需安装 Python 或任何依赖
3. 首次使用创建管理员账号即可

### 方式二：从源码运行
```bash
# 安装依赖
pip install -r requirements.txt

# 运行
python main.py
```

## 📁 项目结构

```
presale-desktop/
├── main.py              # 入口
├── config.py            # 全局配置
├── database.py          # SQLite 数据库层
├── requirements.txt     # Python 依赖
├── ui/
│   ├── login_window.py  # 登录/注册
│   ├── main_window.py   # 主窗口（侧边栏+导航）
│   ├── dashboard.py     # 仪表盘
│   ├── brands_tab.py    # 品牌管理
│   ├── products_tab.py  # 产品管理（含图片上传）
│   ├── orders_tab.py    # 订单管理（含日期选择+状态切换）
│   ├── search_tab.py    # 全局搜索
│   ├── team_tab.py      # 团队管理
│   ├── styles.py        # UI 主题
│   └── components.py    # 可复用组件
├── utils/
│   └── docx_export.py   # Word 导出
└── data/                # 本地数据（不上传）
    ├── presale.db       # SQLite 数据库
    └── images/          # 产品图片
```

## 🛠️ 技术栈

- **Python 3.13** + **customtkinter**（现代 UI）
- **SQLite** 本地数据库
- **python-docx** Word 导出
- **Pillow** 图片处理
- **PyInstaller** 打包为 .exe
